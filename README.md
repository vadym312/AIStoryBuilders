### Data Structure

- **Story**
  - *Title*
  - *Style*
  - *Theme*
  - *Tone*
  
- **Locations**
  - *Location Name*
  - *Description*
  - *StartDate (nullable)*
  - *EndDate (nullable)*
  
- **Characters**
  - *Character Name*
  - *Description*
  - *Goals*
  - *Background*
    - *Paragraphs*
      
- **Paragraphs**   
  - *Content (500 word maximum)*
  - *Characters*
  - *Location*
  - *Timeline*

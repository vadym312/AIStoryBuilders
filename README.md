### Data Structure

- **Story**
  - *Title*
  - *Style*
 
  - **Timelines**
  - *Timeline Name*
  - *Description*
  - *StartDate*
  - *EndDate* (nullable)
  
- **Locations**
  - *Location Name*
  - *Description*
  
- **Characters**
  - *Character Name*
  - *Descriptions*
    - *Desciption Type*
    - *Timeline Name*

- **Chapters** 
  - **Paragraphs**   
    - *Content*
    - *Timeline Name*
    - *Location Name*
    - *Characters*
